# 1. What is Coding?

Coding is the process of converting human logic into instructions a machine can execute.

A program always follows:

**INPUT → PROCESS → OUTPUT**

Example:
- Input: number 5
- Process: multiply by 2
- Output: 10

---

# 2. Core Thinking Model (IMPORTANT)

Before writing code, think like this:

## SYSTEM THINKING MODEL

- DATA → What information exists?
- LOGIC → What rules apply?
- ACTION → What should happen?
- RESULT → What changes?

This is the same structure used in SPD:

**DATA → ALGORITHMS → CODE → COMPUTE**
